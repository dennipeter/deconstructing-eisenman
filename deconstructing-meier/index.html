Deconstructing Richard Meier

This is an artistic deconstruction of Richard Meier's Church of 2000. Created by Dennis Peter, using p5.js.

Touch Interactions

1. Touch and drag to rotate the model.

2. Tilt phone left-right/up-down to move elements.

3. Works best while sitting in an upright position.

4. If tilt functions are not working, check device/browser settings to enable gyroscope/orientation functions.

5. If animations aren't smooth, it's probably time to upgrade to a new phone. =)
